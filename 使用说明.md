# 幕启 MuQi — AI 小说转剧本工具

## 简介

幕启 MuQi 是一个将中文小说自动转换为结构化剧本的工具，支持输出 YAML 格式，适用于竖屏短剧创作。

## 快速开始

### 方式一：使用构建好的版本（推荐）

构建文件已在 `dist/index.html`，您可以：
1. 直接用浏览器打开 `dist/index.html`
2. 或通过本地服务器运行

### 方式二：启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

## 功能特性

### 1. 多章节小说转换
- 自动识别章节标题
- 每个章节作为独立的 act
- 智能分段处理，避免内容截断

### 2. 角色提取
- 自动识别小说中的角色
- 过滤无效词（代词、语气词等）
- 支持别名处理

### 3. 对白提取
- 识别引号内的对话
- 匹配说话人
- 支持不同引号格式

### 4. 场景构建
- 自动生成场景信息
- 支持 YAML 导出
- 剧本预览

## 测试文件

项目包含示例小说 `example_novel.txt`，您可以：

1. 将其内容粘贴到工具中
2. 或直接上传该文件

## 输出说明

### YAML 格式

```yaml
script:
  meta:
    title: "小说标题"
    source: "用户输入"
    format: "竖屏短剧"
    generated_at: "ISO时间"
    stats:
      characters: N
      scenes: M
      dialogues: K
  acts:
    - act_id: 1
      act_title: "章节标题"
      scenes:
        - scene_id: 1
          location: "场景位置"
          time: "时间"
          mood: "氛围"
          scene_note: "旁白"
          characters:
            - name: "角色名"
              role: "角色"
          dialogues:
            - dialogue_id: 1
              type: "spoken"
              speaker: "角色名"
              line: "对话内容"
              action: "动作"
              note: ""
```

## 常见问题

### Q: 如何配置 AI 功能？
A: 在工具界面点击右上角的设置按钮，输入您的 API Key 和 API URL。

### Q: 支持哪些文件格式？
A: 目前支持 .txt 和 .docx 格式，推荐使用纯文本格式。

### Q: 转换精度如何？
A: 使用 AI 模式可以获得更高的精度，规则模式适合快速预览。
