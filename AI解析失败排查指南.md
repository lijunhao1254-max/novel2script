# AI解析失败排查指南

## 📌 如何获取详细错误信息

### 步骤1：打开浏览器控制台
1. 打开浏览器访问 http://localhost:3001/
2. 按下 `F12` 键打开开发者工具
3. 切换到 `Console` 标签页

### 步骤2：重新尝试转换
1. 在文本输入框中粘贴小说内容
2. 点击「AI智能转换」按钮
3. 观察控制台输出

### 步骤3：查看错误信息
控制台会显示类似以下的错误信息：

```
❌ 分段 1 AI 调用异常: API Key 无效或已过期，请重新配置
❌ 分段 2 AI 输出验证失败: YAML 输出为空或过短
   输入长度: 1500 字
   AI返回长度: 0 字
```

---

## 🎯 常见错误原因及解决方案

### 1. API Key 问题（最常见）

**错误信息**:
```
AI 调用失败 (HTTP 401)
API Key 无效或已过期，请重新配置
```

**解决方案**:
1. 访问智谱开放平台: https://open.bigmodel.cn/
2. 登录并获取有效的 API Key
3. 在工具中点击「修改Key」按钮
4. 粘贴新的 API Key（确保没有多余空格）

### 2. 请求超时

**错误信息**:
```
AI 请求超时（60秒），请稍后重试
```

**解决方案**:
1. 检查网络连接是否稳定
2. 尝试缩短文本长度（每次处理5-10章）
3. 等待片刻后重试

### 3. 请求被限流

**错误信息**:
```
AI 请求过于频繁，请稍等后重试
HTTP 429
```

**解决方案**:
1. 等待几分钟后重试
2. 减少单次请求的文本量
3. 避免频繁点击转换按钮

### 4. AI返回内容为空

**错误信息**:
```
AI 输出验证失败: YAML 输出为空或过短
AI返回长度: 0 字
```

**解决方案**:
1. 检查网络连接
2. 检查API Key是否有效
3. 尝试使用规则引擎模式（不配置AI Key）

### 5. AI返回格式不正确

**错误信息**:
```
AI 输出验证失败: 缺少场景或对白结构
```

**解决方案**:
1. 这种情况通常会自动降级到规则引擎
2. 如果频繁出现，建议检查网络稳定性
3. 尝试分批处理文本

---

## 📊 错误类型对照表

| HTTP状态码 | 错误类型 | 解决方案 |
|-----------|---------|---------|
| 401 | API Key无效 | 重新获取API Key |
| 403 | 权限不足 | 检查API Key权限 |
| 400 | 请求参数错误 | 缩短文本长度 |
| 429 | 请求过于频繁 | 等待后重试 |
| 500/502/503 | 服务端错误 | 稍后重试 |
| 超时 | 网络问题 | 检查网络连接 |

---

## ✅ 降级机制说明

当AI解析失败时，系统会自动切换到**规则引擎模式**，确保转换流程不会中断。您看到的输出就是规则引擎生成的结果。

---

## 📝 操作建议

1. **优先检查API Key** - 这是最常见的原因
2. **打开控制台** - 获取详细的错误信息
3. **分批处理** - 超长文本建议分多次转换
4. **尝试规则引擎** - 如果AI一直失败，可以不配置API Key使用规则引擎
